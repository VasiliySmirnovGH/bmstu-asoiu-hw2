## 📌 Описание и Вариант задания

Консольное приложение на языке **C#** (.NET 8), реализующее объектно-ориентированную модель данных со связью «один-ко-многим», сохранением состояния в реляционной базе данных **SQLite** и возможностью генерации отчетов с помощью паттерна **Fluent Interface**.

**Спецификация Варианта №21:**
- **Справочная таблица (сторона «один»):** Микрорайоны (`Microdistrict`)
- **Основная таблица (сторона «много»):** Жилые дома (`House`)
- **Контролируемое числовое поле:** Количество этажей (`floors`) — валидация запрещает отрицательные значения.

---

## ⚙️ Функциональные возможности

1. **Первичная инициализация:** Автоматическое создание таблиц БД и импорт демонстрационных данных из приложенных CSV-файлов при первом запуске.
2. **Интерактивное CRUD-меню:** Полный цикл управления жилыми домами (Создание, Чтение, Обновление, Удаление). Перед добавлением или редактированием автоматически выводится актуальный справочник микрорайонов для удобства привязки внешнего ключа.
3. **Бизнес-валидация:** Проверка корректности данных на уровне свойств классов-моделей (выброс `ArgumentException` при попытке задать отрицательную этажность).
4. **Построитель аналитических отчётов:** Гибкий компоновщик табличных представлений (`ReportBuilder`) на базе интерфейса Fluent Interface.

---

## 📊 Реализованные аналитические отчёты

В программе настроены 3 обязательных аналитических среза с использованием SQL-агрегации:
1. **Ведомость жилых домов по микрорайонам** — прямое связывание таблиц через `JOIN` с лексикографической сортировкой по адресу.
2. **Статистика количества домов по жилым массивам** — группировка `GROUP BY` с подсчетом количества объектов через `COUNT(*)`.
3. **Рейтинг микрорайонов по средней этажности застройки** — группировка `GROUP BY` с вычислением математического ожидания через `AVG(floors)`, округлением до сотых и сортировкой по убыванию.

Для компиляции и запуска приложения вам понадобятся установленный SDK .NET 8 и установленный пакет `Microsoft.Data.Sqlite`.

```bash

git clone [https://github.com/your-username/HousingFundApp.git](https://github.com/your-username/HousingFundApp.git)
cd HousingFundApp

# Восстановление зависимостей и пакета SQLite
dotnet restore

# Сборка проекта
dotnet build --configuration Release

# Запуск консольного приложения
dotnet run --project src/HousingFundApp.csproj
